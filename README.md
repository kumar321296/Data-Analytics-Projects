# Bank Marketing Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Bank Marketing dataset to understand customer behavior and factors influencing term deposit subscription.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Steps
- Data loading and inspection
- Missing value analysis
- Descriptive statistics
- Distribution analysis of target variable
- Correlation analysis using heatmaps

## Key Insights
- Dataset is highly imbalanced (majority customers did not subscribe)
- Economic indicators show strong correlation with subscription behavior
- Dataset is suitable for further predictive modeling
